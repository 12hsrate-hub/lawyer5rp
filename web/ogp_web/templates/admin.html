Advanced / Compatibility

Этот экран остаётся только для raw law diagnostics, legacy/runtime detail и low-level operator сценариев. Основной рабочий путь начинается с /admin/servers и карточки конкретного сервера.

Открыть server workspace
Когда нужен этот экран
Для low-level law diagnostics, raw compatibility checks и legacy/runtime detail, которые не должны мешать обычной настройке сервера.
Что делать обычно
Открывайте /admin/servers, затем карточку нужного сервера и работайте через вкладки Законы, Функции и Шаблоны вывода.
Статус миграции
Повседневные operator flows уже перенесены в server-centric workspace. Этот экран сохраняется как compatibility seam и не должен конкурировать с новым путём.
Чего не делать здесь
Не начинайте здесь новый server setup, не используйте этот экран как primary workspace и не считайте его заменой карточки сервера.
