# Frontend starter templates for login page

В каталоге собраны два стартовых шаблона для быстрого старта страницы логина:

1. `nextjs_login_starter/` — рекомендованный production-путь на Next.js + TypeScript.
2. `reflex_login_starter/` — Python-first вариант на Reflex.

## Как использовать

- Скопируйте выбранный шаблон в отдельный frontend-репозиторий или в папку монорепо.
- Подключите endpoint авторизации текущего backend (`/auth/login` или ваш фактический путь).
- Согласуйте UI/брендинг позже: структура и логика уже заложены.

## Что уже включено

- Базовая форма логина.
- Обработка состояний: loading/success/error.
- Место для интеграции с текущим API и хранения токена.
